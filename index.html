🎯 게임 목표

좌표평면 위에 자신의 픽셀을 놓아 가로, 세로, 대각선 방향으로 4개를 먼저 연결하면 승리합니다. 단, 적의 픽셀을 피하거나 차단하는 전략이 필요합니다!

🔄 게임 진행 순서

  1. 각 플레이어는 아바타와 색상을 배정받습니다.
  2. 가위바위보 또는 주사위로 선공을 결정합니다.
  3. 자신의 턴에 X축 슬라이더와 Y축 슬라이더를 조작해 좌표를 선택합니다.
  4. 선택한 좌표에 픽셀을 놓습니다. (30초 타임리밋)
  5. 이미 픽셀이 있는 좌표는 선택할 수 없습니다.
  6. 4연속 완성 또는 모든 칸을 채우면 게임이 종료됩니다.

📍 좌표란 무엇인가?

좌표는 평면 위의 점의 위치를 나타내는 숫자 쌍입니다. (x, y) 형식으로 씁니다.

📏 축(Axis) 이해하기

  • X축 (가로): 오른쪽 → 양수(+), 왼쪽 → 음수(−)
  • Y축 (세로): 위쪽 → 양수(+), 아래쪽 → 음수(−)
  • 원점: X축과 Y축이 만나는 점 (0, 0)

🗺️ 4개의 사분면

(−, −)

O(0,0)

✏️ 좌표 읽는 법

  1. 먼저 X축 값(가로)을 읽습니다.
  2. 그 다음 Y축 값(세로)을 읽습니다.
  3. 예: 오른쪽으로 3칸, 위로 2칸 → (3, 2)
  4. 예: 왼쪽으로 1칸, 아래로 4칸 → (−1, −4)
(3, 2) → 1사분면 (−2, 3) → 2사분면 (−1, −4) → 3사분면 (4, −1) → 4사분면
